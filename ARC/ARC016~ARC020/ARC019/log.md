# 考察
### A お買い物クライシス
- 愚直な方法で解いたけど、簡単やった。解き方が複数あって前のこういう系統の問題の復習になって良かった。

- 別解１ (`join`を使う方法)
```python
S = input()

dicts = {'O': '0', 'D': '0', 'I': '1', 'Z': '2', 'S': '5', 'B': '8'}
print(''.join(dicts.get(item, item) for item in S))
```

- 別解２ (`str.tranlate`を使う方法)
```python
print(
    input().translate(
        str.maketrans(
            {'O': '0', 'D': '0', 'I': '1', 'Z': '2', 'S': '5', 'B': '8'}
        )
    )
)
```
