- [2点を通る直線から離れた位置にある点までの距離を求める](https://tokibito.hatenablog.com/entry/20121227/1356581559)

```python
# x, y からある直線までの距離を計算する
def cal(x1, y1, x2, y2):
    global x, y

    u = numpy.array([x2 - x1, y2 - y1])
    v = numpy.array([x - x1, y - y1])
    L = abs(numpy.cross(u, v) / numpy.linalg.norm(u))
    
    return L
```
